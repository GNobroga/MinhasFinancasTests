## Objetivo do projeto

Este projeto contém testes unitários, de integração e E2E desenvolvidos para um teste técnico de Desenvolvedor de Testes. Apenas os testes foram incluídos conforme especificado nos requisitos, os demais códigos foram descartados.

- Para os testes da API e instruções de execução, consulte [API Tests](./api/TESTING.md)
- Para os testes Web e instruções de execução, consulte [Web Tests](./web/TESTING.md)

## Tecnologias utilizadas

### Web
- [Playwright](https://playwright.dev/)
- [Vitest](https://vitest.dev/)
- [Faker](https://fakerjs.dev/)

### API
- [xUnit](https://xunit.net/)
- [Moq](https://github.com/moq/moq)
- [Fluent Assertions](https://fluentassertions.com/)
- [ASP.NET MVC Testing](https://learn.microsoft.com/en-us/aspnet/core/test/integration-tests)
- [Bogus](https://github.com/bchavez/Bogus)
